# ZLADIV

## Function Signature

```fortran
ZLADIV(X, Y)
```

## Description


 ZLADIV := X / Y, where X and Y are complex.  The computation of X / Y
 will not overflow on an intermediary step unless the results
 overflows.

## Parameters

### X (in)

X is COMPLEX*16

### Y (in)

Y is COMPLEX*16 The complex scalars X and Y.
